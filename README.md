# Matrix Operations Tool

A GUI-based Matrix Operations Tool built using Python, Tkinter and NumPy.

## Features
- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Transpose
- Matrix Determinant

## Technologies used 
-python
-tkinter
-Numpy

## How to run 
1. open the project in vs code 
2. make sure python and numpy are installed 
3. run 'MatrixOperationsTools.py'.
4. Select the required matrix operation.
5. enter the matrix size and values .
6.  view the results
